# 主板与芯片组技术

## 概述

服务器主板是连接所有硬件组件的核心平台，芯片组则负责协调CPU、内存、存储和I/O设备之间的数据传输。本文档深入解析服务器主板架构、芯片组技术、扩展接口规范及设计原理。

## 服务器主板架构

### 主板规格标准

```yaml
服务器主板规格:
  E-ATX (Extended ATX):
    尺寸: 305mm × 330mm (12" × 13")
    用途: 高端工作站、入门级服务器
    扩展槽: 7-8个
    
  EEB (Enterprise Electronics Bay):
    尺寸: 305mm × 330mm
    用途: 企业级服务器
    特点: 增强型ATX，更多I/O
    
  SSI EEB:
    尺寸: 305mm × 330mm
    标准: Server System Infrastructure
    用途: 标准化企业服务器
    
  SSI CEB:
    尺寸: 305mm × 267mm
    用途: 紧凑型企业服务器
    特点: 比EEB更紧凑
    
  Proprietary (厂商专有):
    特点: 非标准尺寸
    用途: 特定机箱设计
    厂商: Dell, HP, 浪潮等
```

### 内存配置规则

```yaml
服务器内存配置最佳实践:
  通道平衡:
    原则: 所有通道安装相同数量、相同规格DIMM
    影响: 不平衡配置导致带宽降级
    
  Rank配置:
    单Rank (1R):
      优点: 最低延迟
      缺点: 容量受限
      适用: 延迟敏感应用
      
    双Rank (2R):
      优点: 容量翻倍
      缺点: 略增延迟
      适用: 一般应用
      
    四Rank (4R):
      优点: 最大容量
      缺点: 延迟增加
      适用: 内存密集应用
      
  安装顺序:
    原则: 从远离CPU的槽位开始安装
    原因: 信号完整性、布线优化
    
  混插规则:
    容量: 可混插不同容量
    速度: 按最低速度运行
    厂商: 建议相同厂商
    类型: 禁止RDIMM/LRDIMM混插
```

### 选型考量因素

```yaml
主板选型清单:
  CPU兼容性:
    - 插槽类型
    - 支持的CPU系列
    - 最大TDP支持
    - BIOS更新需求
    
  内存支持:
    - 最大容量
    - 插槽数量
    - 支持类型 (RDIMM/LRDIMM)
    - 最高频率
    - ECC支持
    
  扩展能力:
    - PCIe插槽数量和规格
    - CPU直连 vs PCH连接
    - OCP网卡支持
    - GPU支持数量
    
  存储接口:
    - NVMe槽位数
    - SATA端口数
    - SAS控制器
    - RAID支持
    
  网络集成:
    - 板载网卡规格
    - 管理网口
    - OCP 3.0支持
    
  管理功能:
    - BMC芯片
    - TPM支持
    - 远程管理能力

    - 供电相数

    - 保修政策
```
